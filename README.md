## Description
This project is a simple **Rock Paper Scissors game** built using **HTML, CSS, and JavaScript**. The game allows a user to play against the computer, where the computer randomly selects Rock, Paper, or Scissors. Based on the standard game rules, the result (Win, Lose, or Draw) is displayed instantly.

This project is beginner-friendly and helps in understanding core JavaScript concepts such as **event handling, conditional logic, random number generation, and DOM manipulation**.

---

## Features
- User vs Computer gameplay
- Random computer choice generation
- Instant result display
- Simple and interactive UI
- Beginner-friendly project
